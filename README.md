# [deployable-log](https://github.com/deployable/deployable-log)

Deployable Logging

### Install
 
    npm install deployable-log --save

    yarn add deployable-log

### Usage

```javascript

const deployable-log = require('deployable-log')
deployable-log.demo()

```
